# Task-Management-System

## 📌 Project Description
The Task Management System is a web-based application developed using Angular. It allows users to manage and track tasks efficiently through a simple and interactive interface.

The application enables users to add employee, assign tasks, update task status, and delete records. It also supports data persistence using browser Local Storage and provides an option to export data into an Excel-compatible format.

---

## 🚀 Features
- Add employee with task details
- Display employee data in table format
- Update task status (Pending / Completed)
- Delete employee records
- Data persistence using Local Storage
- Export data to Excel (CSV file)
- Responsive and user-friendly UI

---
